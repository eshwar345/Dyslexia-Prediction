# Dyslexia Prediction using Machine Learning

## Overview
Dyslexia is a learning disorder that affects reading, writing, and spelling abilities due to differences in how the brain processes language :contentReference[oaicite:0]{index=0}.  
This project aims to predict the likelihood of dyslexia using machine learning techniques based on input features related to cognitive and linguistic patterns.

---

## Problem Statement
Early detection of dyslexia is important but challenging.  
This project builds a predictive model to identify potential dyslexia cases using structured data, helping in early screening and intervention.
